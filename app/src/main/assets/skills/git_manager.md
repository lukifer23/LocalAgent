# Git Manager

This skill allows you to manage git repositories within the Termux environment.

## Instructions
- Use `git status` to see pending changes.
- Use `git diff` to review changes before committing.
- When committing, always provide a clear and concise commit message.
- You can manage branches with `git branch` and `git checkout`.

## Available Resources
- `git status`: Show the working tree status.
- `git diff [filename]`: Show changes in files.
- `git add [filename]`: Add file contents to the index.
- `git commit -m "[message]"`: Record changes to the repository.
- `git log -n [count]`: Show commit logs.
- `git branch`: List branches.
- `git checkout [branch]`: Switch branches or restore working tree files.
