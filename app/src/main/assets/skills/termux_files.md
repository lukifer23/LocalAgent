# Termux File Manager

This skill provides direct access to the Termux file system.

## Instructions
- Use `ls` to browse the current directory.
- Use `cat` to read a file's content.
- You can create directories with `mkdir` to organize your work.

## Available Resources
- `ls -F`: List files and directories.
- `cat [filename]`: Read the contents of a file.
- `mkdir -p [dir]`: Create a directory path.
- `rm [filename]`: Delete a file.
